# AI Prompt Engineering Skill

- Sempre formatar saídas como JSON válido usando blocos Markdown ou definindo `response_format`.
- Injetar o contexto completo do projeto e evitar suposições.
- Criar validações robustas nas instruções para prevenir campos soltos ou faltando.
