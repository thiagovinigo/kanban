# React Glassmorphism Skill

- Use blocos semitransparentes com fundo branco ou preto: `rgba(255, 255, 255, 0.1)`.
- Adicione desfoque de fundo: `backdrop-filter: blur(10px)`.
- Utilize bordas finas com pouca opacidade: `border: 1px solid rgba(255, 255, 255, 0.2)`.
- Evite criar estilos dependentes de classes utilitárias complexas como Tailwind se não estiver explicitamente configurado. Use objetos de estilo inline ou `index.css`.
